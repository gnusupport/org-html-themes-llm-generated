## Usage

### Local Preview:
```bash
# In your Org file:
#+SETUPFILE: org/html-theme-baby-local.setup

# Build:
emacs --batch --visit=your-file.org --funcall org-html-export-to-html
```

### Remote Use (GitHub Pages):
```bash
# In your Org file:
#+SETUPFILE: org/html-theme-baby.setup

# The CSS loads from:
# https://fniessen.github.io/org-html-themes/css/baby_theme/css/baby.css
```
