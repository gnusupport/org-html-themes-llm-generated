** Section 4: Code Examples

Here's a Python example with syntax highlighting:

#+BEGIN_SRC python
class MagazineTheme:
    """A modern theme for Org HTML exports."""
    
    def __init__(self, name="Magazine"):
        self.name = name
        self.features = [
            "Two-column layout",
            "Beautiful typography",
            "Responsive design"
        ]
    
    def render(self, content):
        """Render content with theme styling."""
        return f"<article class='{self.name}'>{content}</article>"
#+END_SRC
