* Overview

This document demonstrates the /Personal Notes/ theme, designed for maximum readability with:

- Optimal line length (65-75 characters)
- Large, comfortable body font (18px)
- Excellent line height (1.7)
- High contrast text (WCAG AA compliant)
- Minimal distractions, pure content focus

* Section 1: Typography and Reading Experience

The Personal Notes theme prioritizes comfortable, extended reading sessions. The typography has been carefully calibrated for optimal comprehension and minimal eye strain.

** Headings Hierarchy

The heading system provides clear visual hierarchy while maintaining consistency:

*** Level 3 Heading

This is a level three heading. Notice how the left border accent helps you quickly identify heading levels while keeping the design clean and minimal.

**** Level 4 Heading

***** Level 5 Heading

** Paragraphs and Text Flow

Good typography is invisible. The Personal Notes theme lets your words shine through with:

- Optimal line height of 1.7 for comfortable eye tracking
- Maximum line length of 70ch (characters) for easy line-to-line return
- High contrast text (#1A1A1A on #FAFAFA) exceeding WCAG AA requirements
- Clean, modern sans-serif font stack

This paragraph demonstrates the comfortable reading experience. Notice how your eyes naturally flow from the end of one line to the beginning of the next without strain.

The line length has been carefully calibrated based on readability research. Lines that are too long cause eye fatigue, while lines that are too short disrupt reading rhythm. The sweet spot of 65-75 characters provides optimal comprehension.

** Emphasis and Formatting

Text can be emphasized in several ways:

*Strong text* for important points that need extra attention.

/Italic text/ for subtle emphasis, technical terms, or foreign words.

_Underlined text_ for special cases (though links are preferred).

=Code-like text= for inline code or technical references.

You can combine multiple emphasis marks: /=bold italic=/ or _//underline italic//_.

#+BEGIN_SRC python
def fibonacci(n):
    """Generate Fibonacci sequence up to n."""
    if n <= 0:
        return []
    elif n == 1:
        return [0]
    
    sequence = [0, 1]
    while sequence[-1] + sequence[-2] <= n:
        sequence.append(sequence[-1] + sequence[-2])
    
    return sequence

print(fibonacci(100))
#+END_SRC

* Section 10: Responsive Design

The Personal Notes theme adapts beautifully to different screen sizes:

** Desktop Experience

On desktop monitors, content is centered with optimal line length for comfortable reading. Margins provide breathing room, and the clean layout keeps focus on the content.

** Tablet Experience

On tablets, the layout adjusts slightly while maintaining readability. The line length remains optimal, and touch targets are appropriately sized.

** Mobile Experience

On mobile devices, the font size adjusts slightly, and padding is reduced to maximize content area while maintaining comfort. The single-column layout is ideal for small screens.
