4. 💻 Code Examples

Code blocks feature aurora-colored syntax highlighting with flowing top borders.

4.1. JavaScript Example**

// Aurora gradient configuration
const auroraColors = {
  deep: '#0F0C29',
  mid: '#302B63',
  light: '#24243E',
  cyan: '#00D4FF',
  purple: '#9D4EDD',
  pink: '#FF006E'
};

function createAuroraGradient(element) {
  const gradient = `linear-gradient(135deg, 
    ${auroraColors.deep} 0%, 
    ${auroraColors.mid} 40%, 
    ${auroraColors.light} 80%)`;

  element.style.background = gradient;
  element.style.backgroundSize = '400% 400%';
  element.style.animation = 'aurora-flow 15s ease infinite';
}

4.2. Python Example**

"""
Aurora Plain Theme - CSS Configuration
Minimal design with flowing lights aesthetic
"""

from dataclasses import dataclass
from typing import List

@dataclass
class AuroraPalette:
    """Aurora color palette"""
    background_deep: str = "#0F0C29"
    background_mid: str = "#302B63"
    background_light: str = "#24243E"
    cyan: str = "#00D4FF"
    purple: str = "#9D4EDD"
    pink: str = "#FF006E"

    def to_css_gradient(self) -> str:
        """Generate CSS gradient string"""
        return f"linear-gradient(135deg, {self.background_deep} 0%, {self.background_mid} 40%, {self.background_light} 100%)"

# Create aurora theme instance
theme = AuroraPalette()
print(f"Background: {theme.to_css_gradient()}")

8. 📋 Notes

8.1. About the Design**

The Aurora Plain theme captures the ethereal beauty of northern lights in a digital format. The flowing gradient background creates a sense of movement, while the carefully chosen color palette evokes the mystical aurora borealis.

8.2. Key Features **

  • Minimal design without sidebar
  • Flowing gradient background animation
  • Aurora-gradient text for headings
  • Soft glow effects on interactive elements
  • Responsive layout for all screen sizes

8.3. Fonts Used **

  • Headings: Orbitron (futuristic sans-serif)
  • Body: Exo 2 (tech-inspired readable font)
  • Code: Fira Code (programmer-friendly monospace)

9. 🔬 Technical Details

9.1. CSS Structure **

The theme uses CSS custom properties (variables) for consistent theming. All colors, spacing, and effects are centralized for easy customization.

9.2. Animations **

  • Background gradient flows over 15 seconds
  • Text gradient animation on H1 (4s infinite)
  • Line glow animation under title (3s ease-in-out)
  • TODO task pulse effects (2.5s infinite)
  • Code block border flow (3s infinite)

9.3. Performance **

Animations use GPU-accelerated properties (transform, opacity) where possible. The theme respects `prefers-reduced-motion` for users who prefer less animation.
