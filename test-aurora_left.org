** CSS Grid Layout Alternative Note **
This theme avoids CSS Grid as per project standards, using flexbox for the sidebar layout instead.

#+BEGIN_SRC css
/* Aurora Left Sidebar Layout */
#container {
  display: flex;
  min-height: 100vh;
  position: relative;
}

#table-of-contents {
  position: fixed;
  left: 0;
  top: 0;
  width: 300px;
  height: 100vh;
  background: linear-gradient(180deg, 
    rgba(15, 12, 41, 0.98) 0%, 
    rgba(48, 43, 99, 0.95) 40%, 
    rgba(123, 44, 191, 0.9) 100%);
  overflow-y: auto;
}

#content {
  margin-left: 300px;
  padding: 3.5rem 2.5rem;
  max-width: calc(100% - 300px - 4rem);
}
#+END_SRC

** JavaScript for Active Section Detection **
#+BEGIN_SRC javascript
// Highlight active section in sidebar
function highlightActiveSection() {
  const sections = document.querySelectorAll('#content h2, #content h3');
  const navLinks = document.querySelectorAll('#table-of-contents a');
  
  sections.forEach(section => {
    section.addEventListener('scroll', () => {
      const rect = section.getBoundingClientRect();
      
      if (rect.top <= 150 && rect.bottom >= 150) {
        navLinks.forEach(link => link.classList.remove('active'));
        const activeLink = document.querySelector(`#table-of-contents a[href="#${section.id}"]`);
        if (activeLink) activeLink.classList.add('active');
      }
    });
  });
}

// Initialize on page load
document.addEventListener('DOMContentLoaded', highlightActiveSection);
#+END_SRC

* 📋 Design Notes
:PROPERTIES:
:CATEGORY: Reference
:END:

** Layout Philosophy **
The left sidebar design prioritizes easy navigation while maintaining the beautiful aurora aesthetic. Users can always see where they are and how to move through the content.

** Key Differences from Plain Theme **
1. Sidebar with fixed 300px width
2. Vertical aurora gradient in sidebar
3. Active section highlighting
4. Content offset from sidebar
5. Responsive top bar on mobile

** Typography Consistency **
Both Aurora themes use the same font stack:
- Orbitron for headings (futuristic)
- Exo 2 for body text (tech-readable)
- Fira Code for code (programmer-friendly)

** Color Palette **
Both themes share the same aurora colors:
- Deep background: #0F0C29 → #302B63 → #24243E
- Cyan accent: #00D4FF
- Purple accent: #9D4EDD
- Pink accent: #FF006E

* 🔬 Technical Implementation
:PROPERTIES:
:CATEGORY: Reference
:END:

** Flexbox Layout **
The theme uses CSS Flexbox for the sidebar-content container, avoiding CSS Grid as per project standards.

** Fixed Positioning **
The sidebar uses fixed positioning to stay visible while scrolling through long documents.

** Responsive Transform **
At 768px breakpoint, the sidebar transforms from a left column to a top navigation bar, maintaining usability on mobile devices.

** Performance Considerations **
- GPU-accelerated animations
- Minimal repaints on scroll
- Optimized gradient rendering
- Respects prefers-reduced-motion
